Research and Projects
Master Research Project (Ongoing)
Morphological Characterisation of the Human Knee Joint X-ray images.

Extract the interested morphological characterisation of the Human Knee Joint X-ray images

  • Supervisor: Dr. Nicolo Malagutti.
  • Bone Segmentation: A coarse to fine model structure which predict high resolution mask by point-render to save memory.
  • Signed Distance supervision: - A SDF-head, which introduce geometry information for segmentation network.
  • Postprocessing Algorithm: A postprocessing Algorithm which is loyal to the definition of morphological characterisation
  • Impact (Currently): The mean squared error of HKA between the algorithm prediction and doctors' record (ground truth) decreases from 1.83 (vanilla U-Net) to 1.32.
Generalised Novel View Synthesis
A Generalised Novel View Synthesis based on Multi-view stereos.
  • Feature Interpolation Strategy: For every sample points on rays, interpolates features with nearest depth values.
  • Opacity Via TSDF: Get the initial opacities of sample points via Laplacian distribution of TSDF with their nearest depth, and refine them by MLP
  • Feature Merger: Only keep the features of the source view with the largest opacity value
HOI Detection
A Human-Object-Interaction detector based on HOI Transformer
  • Adaptive Queries: A grouping based query block to get adaptive queries for single images for decoding HOI instances.
  • Multi-Modal Information: Fusing queries with the word vectors pretrained from action verbs in training dataset, and a action represent head to predict action embeddings.
  • Impact: The mAP shows some degree of increase on Rare examples (The action rarely happened in the dataset) on HICO-DET dataset. Get High Distinct (90/100)

Lane Detection
A Lane detection alogrithm based on Bezier curve modeling
  • Large Kernel Enhanced Attention Module: A Large Kernel Enhanced Attention module, which is used to aggregate spatial information for the transformer.
  • Random Lane masking: A Random Line Masking augmentation module to mimic lane broken.
  • Impact: achieves very competitive performances on several benchmarks, i.e. No.4 on the LLAMAS test set and No.1 on the CULane and the TuSimple dataset among all curve-based methods. Get High Distinct score (82/100)

Internship in SenseTime Research
A Table recognition Algorithm based on Location Classification
  • Decoding table structure via Position Vector: Proposing a cell position representation based on 4-dim fourier transform, using it to decode the logical position of each cell.
  • Multi-task supervision:Supervising the cell location by cross-Entropy loss and weighted score regression loss
  • Postprocessing Algorithm:A postprocessing algorithm which generate the table lines based on model prediction by recursive allignment.
  • Impact: The TEDs score of table recognition increase from 94.26% to 95.82%
